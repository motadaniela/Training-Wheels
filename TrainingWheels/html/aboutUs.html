Sobre este projeto

Este projeto foi criado no âmbito de um projeto interdisciplinar, e envolve as cadeiras de Projeto I, POO, IDA e CPM, para cada qual desenvolvemos diferentes tipos de conteúdos.
O objetivo desta plataforma é, sobretudo, incentivar futuros condutores a estudar o código da estrada, sem serem desencurajados pelas aulas de código.
